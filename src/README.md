## File structure

- _main_ - the main file containing setup and run functions. In this file, the program begins.
- _global-var_ - contains all global variables and imported libraries.
- _classes_ - contains all class models.
- _ai_ - contains all AI and ML data, such as training models, teaching data access or weights output.
- _flask-app_ = contains backend functions responsible for setting up the website and server connections.
- _dl-data_ - responsible for download functions (from the Spotify API to the user), such as requesting the API for accessing the user listening history.
- _ul-data_ - responsible for upload functions (from the user to the Spotify API), such as saving the generated playlist on the user account.
